# Gamify IAS UPSC AI

This repository contains the Gamify IAS UPSC AI frontend.

Quick start

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

Replace the app icon and metadata in `index.html` (project root) and add your favicon in the `public/` folder.

If you need help wiring a custom logo into the UI, tell me where you'd like it shown and I will add a placeholder and instructions.
